# v0.16_2 策略版本说明

## 1. 文件定位

- 策略代码：`策略复现与回测/策略代码/资产风险平价策略0.16_2（信号条件化风险预算）.py`
- 版本号：`0.16_2`
- 基础版本：`0.16`
- 主要目标：在 `v0.16` 周频风险平价框架下，将股指期货信号从“外部固定仓位模块”改为“风险预算调节因子”，让 `沪深300主连`、`中证1000主连` 与其他资产共同进入风险平价优化器。

## 2. 核心变化

`v0.16` 的股指期货处理方式是先根据信号确定一个独立股指仓位，再把剩余仓位交给非股指风险平价模块。`v0.16_2` 改为同池优化：

- `沪深300主连`、`中证1000主连` 放回风险平价资产池。
- 所有已上市资产共同使用过去 12 个月收益估计 EWMA 半协方差。
- 风险平价优化从等风险预算改为带预算向量的风险预算平价。
- 股指期货信号只调整两只股指期货的目标风险预算，不直接指定资金仓位。

风险预算目标函数为：

```text
0.5 * x.T @ cov @ x - sum(b_i * log(x_i))
```

其中 `b_i` 为归一化后的目标风险预算。等预算风险平价是该模型在 `b_i` 全部相等时的特例。

## 3. 股指信号预算规则

每个周度观察日先确定当期已上市资产，然后生成风险预算向量：

- 每个已上市资产原始风险预算默认为 `1.0`。
- 若资产为 `沪深300主连` 或 `中证1000主连`，原始预算乘以股指信号倍率。
- `signal <= 0`：股指风险预算倍率为 `0.5`。
- `0 < signal <= 1`：股指风险预算倍率为 `1 + signal`。
- `signal > 1`：股指风险预算倍率封顶为 `2.0`。
- 所有已上市资产的原始风险预算统一归一化为和为 `1.0` 的目标风险预算。

因此，信号关闭时并不会完全剔除股指期货，而是降低其目标风险贡献；信号打开时则提高其目标风险贡献。其他资产的目标风险预算通过归一化自动对应调整。

## 4. 收益、费用与指标口径

`v0.16_2` 保留 `v0.16` 的回测基础口径：

- 权重估计收益：`数据/JYDB数据替换/日涨跌幅_填充.csv`
- 交易收益：`数据/JYDB数据替换/日涨跌幅_未填充.csv`
- 股指期货信号：`数据/原始数据/股指期货信号.xlsx`
- 周频调仓：每周最后一个实际交易日为观察日，下一个交易日执行目标权重。
- 调仓成本：`FEE_RATE = 0.0005`
- 闲置资金收益：按 `一天期国债逆回购` 年化利率折算，并扣除 `REPO_FEE_RATE = 0.000001`
- 夏普比率：继续使用 `RF=0` 口径。

从结果看，`v0.16_2` 更偏风险调整后的稳健版本：相较 `v0.16`，收益弹性下降，但波动和回撤明显下降，夏普从 `1.68` 提高到 `1.74`。

## 7. 扩展敏感性测试

敏感性测试文件：`策略复现与回测/策略测试代码/资产风险平价策略0.16_2（风险预算参数敏感性测试）.py`

- 低信号倍率：`0.25`、`0.50`、`0.75`
- 正信号满值倍率：`1.50`、`2.00`、`2.50`、`3.00`、`4.00`、`5.00`、`6.00`、`7.00`

仓位接近时，`v0.16_2` 的收益和夏普均低于 `v0.16`。这说明 `v0.16` 的优势不只是来自股指平均仓位更高，还来自其信号规则在股指开仓阶段更集中地配置资金仓位；`v0.16_2` 即使通过风险预算倍率把平均股指仓位抬到接近 `v0.16`，风险预算优化仍会保留更分散的风险结构，收益弹性较弱。
