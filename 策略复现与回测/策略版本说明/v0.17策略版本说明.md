# v0.17 策略版本说明

## 1. 文件定位

- 策略代码：`策略复现与回测/策略代码/资产风险平价策略0.17（周频调仓+股指信号+三策略）.py`
- 版本号：`0.17`
- 基础版本：`0.15`、`0.16`
- 主要目标：在 `v0.16` 股指期货信号和周频调仓框架下，恢复 `v0.15` 的三条策略，并避免股指期货在信号模块和宏观象限模块中重复配置。

## 2. 相对 v0.15 和 v0.16 的核心变化

- 相对 `v0.15`：
  - 调仓频率从月频改为周频。
  - `沪深300主连`、`中证1000主连` 从风险平价资产池和宏观象限资产池中抽离。
  - 股指期货仓位由 `股指期货信号.xlsx` 控制，基准仓位为 `30%`。
- 相对 `v0.16`：
  - 恢复三条策略：`风险平价策略`、`全天候策略`、`全天候增强策略 (动量择时)`。
  - 全天候和增强策略只在非股指剩余资产中运行宏观象限逻辑。
  - 输出图表恢复三条策略净值和三条策略大类仓位，同时新增股指信号与股指仓位曲线。

## 3. 资产池与策略逻辑

股指期货模块：

- `沪深300主连`
- `中证1000主连`

非股指资产池：

- 股票类：`红利低波ETF`
- 债券类：`10年国债主连`、`30年国债主连`
- 商品类：`沪铜主连`、`沪铝主连`、`PTA主连`、`原油主连`、`豆粕主连`
- 黄金类：`沪金主连`

宏观象限已剔除股指期货：

- `增长超预期`：商品类资产
- `增长不及预期`：债券、黄金
- `通胀超预期`：黄金、商品类资产
- `通胀不及预期`：债券、黄金、红利低波ETF

每个周度观察日先计算共同股指期货仓位，再将剩余仓位分配给三条策略的非股指目标权重：

- 风险平价策略：非股指资产按 EWMA 半协方差风险平价分配。
- 全天候策略：先构造非股指宏观象限组合，再对有效象限做风险平价分配。
- 全天候增强策略：根据增长/通胀预期动量选择非股指象限，再对选中象限做风险平价分配。

## 4. 股指期货信号规则

- 股指期货基准仓位为 `30%`。
- 信号 `<=0` 时，股指期货总仓位为 `0`。
- 信号 `0 < signal <= 1` 时，股指期货总仓位为 `30% * signal`。
- 两只股指期货均可交易时在股指模块内等权。
- `中证1000主连` 未上市前，由 `沪深300主连` 单独承担股指期货仓位。

## 5. 收益、费用与指标口径

- 权重估计收益使用 `JYDB主力日涨跌幅_填充.csv`。
- 实际交易收益使用 `JYDB主力日涨跌幅_未填充.csv`。
- 调仓日扣除换手成本，费率为 `FEE_RATE = 0.0005`。
- 闲置资金收益基于 `一天期国债逆回购` 年化利率折算，并扣除 `REPO_FEE_RATE`。
- 夏普比率继续使用 `RF=0` 口径，即 `mean(daily_return) * 252 / (std(daily_return) * sqrt(252))`。
